AI-Prowler Remote Access — Subscription Information
=====================================================

AI-Prowler is free to use with Claude Desktop on your PC. No
subscription is required for local desktop use.

If you want remote access to your AI-Prowler knowledge base from
a phone, tablet, or other computer, you can subscribe for mobile
access at the pricing below. You will also need a Claude Pro
subscription (separate, from Anthropic) to use HTTP MCP support
with Claude.


WHAT IS REMOTE ACCESS?
-----------------------
Remote Access lets you connect your AI-Prowler knowledge base to
Claude.ai and other MCP-compatible AI tools from anywhere over
the internet, using your phone, tablet, or any computer.

Without a subscription, AI-Prowler still works fully as a desktop
application. If you subscribe and later decide you don't need
remote access, you can cancel anytime and continue using
AI-Prowler with Claude Desktop on your PC.


SUBSCRIPTION PLAN
------------------
  AI-Prowler Remote Access  —  $10 / month per user

  Includes:
    - Remote access from any device with a web browser
    - 1 active connection per user

WHAT HAPPENS AFTER PAYMENT
---------------------------
  1. After payment, you will receive a LICENSE KEY by email.
  2. Open AI-Prowler → Settings → Remote Access.
  3. Paste the license key into the 'License key' field.
  4. Restart AI-Prowler's HTTP server — remote access is enabled.
  5. No other changes needed on your machine.

  Important: Your Bearer token stays on your own machine and is
  never sent to anyone. Only the license key issued by AI-Prowler
  is needed to activate your subscription.


GRACE PERIOD
------------
  If your subscription lapses, you have a 30-day grace period
  during which remote access continues to work with a warning
  banner shown on the login page. After 30 days, remote access
  is suspended until renewal.


DESKTOP FEATURES
----------------
  Your AI-Prowler desktop application continues to work fully
  regardless of your remote access subscription status. Indexing,
  searching, OCR, and local Claude Desktop integration all remain
  free.

  Note: Claude (the LLM itself) requires a separate Claude Pro
  subscription (~$20/month from Anthropic) to support remote
  HTTP MCP port operation.
