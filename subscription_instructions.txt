AI-Prowler Remote Access — Subscription Information
=====================================================

AI-Prowler is free to use with Claude Desktop on your PC. No
subscription is required for local desktop use. Indexing, searching,
OCR, and local Claude Desktop integration all remain free regardless
of your subscription status.

If you want remote access to your AI-Prowler knowledge base from
a phone, tablet, or other computer, you'll need an AI-Prowler
Remote Access subscription. You will also need a Claude Pro
subscription (separate, from Anthropic) to use HTTP MCP support
with Claude.

This popup only appears when you click ▶ Start HTTP Server in
Settings → 📡 Remote Access — if you only use AI-Prowler with Claude
Desktop on this PC, you'll never see it.


WHAT IS REMOTE ACCESS?
-----------------------
Remote Access lets you connect your AI-Prowler knowledge base to
Claude.ai and other MCP-compatible AI tools from anywhere over
the internet, using your phone, tablet, or any computer.

Without a subscription, AI-Prowler still works fully as a desktop
application. If you subscribe and later decide you don't need
remote access, you can cancel anytime and continue using
AI-Prowler with Claude Desktop on your PC.


SUBSCRIPTION PLANS
--------------------
  AI-Prowler Home Remote Access  —  $10 / month per user

  Includes:
    - Remote access from any mobile device with the Claude.ai app
    - 1 active connection per user
    - Cancel anytime, no penalty

  AI-Prowler Business  —  $20 / month per user

  Includes:
    - AI-Prowler Personal with Remote Access from any mobile device
    - Server has Remote Access from any mobile device
    - 1 registered domain from Cloudflare with up to 50 tunnel
      connections
    - Per-user role and scope control, with email support for
      field crews
    - Cancel anytime, no penalty


NEW SUBSCRIBER — HOW TO SIGN UP
----------------------------------
Everything is automated — no email, no waiting.

  1. Go to Settings → 📡 Remote Access
  2. Click 🛒 Subscribe — Personal (or 🛒 Subscribe — Business)
  3. Your browser opens a secure Stripe checkout page — complete
     payment there
  4. You'll receive an ACTIVATION CODE by email within a few minutes
  5. Paste the activation code into Settings and click
     ⚡ Configure Mobile Access
     → your Cloudflare tunnel is provisioned and started
       automatically, no separate Cloudflare account needed
  6. Click 📖 Connect Claude.ai (auto) to finish connecting Claude.ai


ALREADY SUBSCRIBED? — RENEW, CANCEL, OR UPDATE
--------------------------------------------------
Use the Manage Subscription → button in Settings → 📡 Remote Access.

This opens a secure, pre-authenticated Stripe-hosted page where you
can:
  - Update your payment method
  - Cancel your subscription
  - Change your Business seat count
  - Review billing history and renewal date

No password or extra login needed — the button opens your account
directly.


GRACE PERIOD
------------
If your subscription lapses, you have a 30-day grace period during
which remote access continues to work with a warning banner shown
in the app. After 30 days, remote access is suspended until you
renew via Manage Subscription →.


DESKTOP FEATURES
----------------
Your AI-Prowler desktop application continues to work fully
regardless of your remote access subscription status. Indexing,
searching, OCR, and local Claude Desktop integration all remain
free.

Note: Claude (the LLM itself) requires a separate Claude Pro
subscription (~$20/month from Anthropic) to support remote
HTTP MCP port operation.


QUESTIONS OR ISSUES?
----------------------
If an activation code doesn't arrive, or Manage Subscription → isn't
working for you, visit https://www.ai-prowler.com for support.
